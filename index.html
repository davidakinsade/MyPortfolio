Final Year Project.

Application that Detects Viruses and Illnesses of an X-Ray with Machine Learning

Project and Thesis: (1:1)

For my final year project, I focused on the method of using X-rays which are used to assess medical conditions with images.

With these images, my idea was to create an application that gives an approach of using Machine Learning algorithms to classify those images. This ensures that the application detects viruses and illnesses of a person to make it easier and quicker to notify and tackle them.

The foundational aspect of creating the application would consist of creating and building a machine learning model.

The Image Dataset.

The image dataset was divided into three sets with five classes each.

(Normal, COVID, Pneumonia, Tuberculosis, Unspecified).

Training Set

Includes the data collection from which the machine learning model will assess (Holds 70-80% of data).

Testing Set

Done to evaluate the overall performance of the model. (15-20% of data).

Validation Set

Used to determine how well the model performed during training (10% of data).

Convolutional Neural Networks.

Creating the Machine Learning Model:

The creation of the model consisted of adding different convolutional layers to the neural network. It was done using the convolutional neural network and building the CNN model.

Convolutional neural networks are employed for image classification. TensorFlow offers resources and libraries for creating and developing CNNs and other deep learning models.

Layers

Input Layer: where the CNN receives its data. The tasks normally consist of dimensions, patterns and channels.

Images contain three colour channels, which would be the ‘RBG’ channel (red, green, blue).

Fully Connected Layers: the final classification task. To create the final output, a flattened feature map is applied from the last convolutional/pooling layer.
